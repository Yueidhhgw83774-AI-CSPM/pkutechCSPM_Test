### 🐛 検出された問題

#### 1. ソースコードのバグ（CSPM-E11）

**場所**: `app/cspm_plugin/router.py:136`

**問題**: 
```python
print(
    f"Error during policy validation tool execution: {e}", exc_info=True
)
```

`print()` は `exc_info` パラメータをサポートしていない。`logger.error()` を使うべき。

**修正方法**:
```python
logger.error(
    f"Error during policy validation tool execution: {e}", exc_info=True
)
```

**影響**: 検証ツール実行時の例外で `TypeError` が発生し、正しいエラーレスポンスが返されない。

#### 2. 既知の脆弱性（CSPM-SEC-02） - xfail

**場所**: `app/cspm_plugin/router.py:105`

**問題**: 
```python
error=f"Internal server error [Error ID: {error_id}]: {str(e)}"
```

`str(e)` が直接レスポンスに含まれるため、内部エラー詳細（IP、ポート等）がクライアントに露出する。

**修正方法**:
```python
error=f"Internal server error. Please contact support with Error ID: {error_id}"
```

**影響**: セキュリティリスク — 内部情報漏洩。
