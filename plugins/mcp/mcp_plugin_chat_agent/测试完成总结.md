## ⚠️ 已知问题

### MCPCA-SEC-04: 状态错误凭证暴露

**问题**: 当前实现（Line 102）直接将 `state["error"]` 返回给用户，可能暴露敏感信息。

**代码位置**: `chat_agent.py:102`

```python
# 当前实现
if error:
    return f"申し訳ありません。処理中にエラーが発生しました。\n\nエラー: {error}", progress
```

**建议修复**:

```python
# 推荐修复
if error:
    error_id = uuid.uuid4()
    logger.error(f"エラー [ID: {error_id}]: {error}")
    return f"申し訳ありません。処理中にエラーが発生しました。[エラーID: {error_id}]", progress
```
