### 主要修复内容

#### 1. UnicodeDecodeError (pickle vs JSON)
**问题**: checkpoint 数据使用 `pickle.dumps()` 而不是 JSON
**修复**:
```python
# 错误
checkpoint_data = b'{"key": "value"}'

# 正确
import pickle
checkpoint_data = pickle.dumps({"key": "value"})
```

#### 2. AttributeError (get_db_pool 路径错误)
**问题**: `get_db_pool` 在 `repository.py` 而不是 `message_converter.py`
**修复**:
```python
# 错误
with patch("app.mcp_plugin.sessions.message_converter.get_db_pool", ...)

# 正确
with patch("app.mcp_plugin.sessions.message_converter.get_async_checkpointer", ...)
```

#### 3. PostgreSQL Mock 结构错误
**问题**: `mock_postgres_pool` 的结构与实际的 psycopg API 不匹配
**修复**: 创建了 `create_pg_mock()` helper function
```python
def create_pg_mock(fetchall_return=None, fetchone_return=None):
    """正确的 PostgreSQL async mock 结构"""
    mock_conn = AsyncMock()
    mock_cursor = AsyncMock()
    # 正确设置 __aenter__/__aexit__
    mock_cursor.__aenter__ = AsyncMock(return_value=mock_cursor)
    mock_cursor.__aexit__ = AsyncMock(return_value=None)
    mock_conn.cursor = MagicMock(return_value=mock_cursor)
    mock_conn.__aenter__ = AsyncMock(return_value=mock_conn)
    mock_conn.__aexit__ = AsyncMock(return_value=None)
    mock_pool = MagicMock()
    mock_pool.connection = MagicMock(return_value=mock_conn)
    return mock_pool
```

#### 4. 模块路径修复
- `get_latest_checkpoint` 在 `repository.py` 而不是 `routes.py`
- `update_checkpoint_metadata` 在 `repository.py`
- `get_async_checkpointer` 是正确的 API (不是 `get_db_pool`)

#### 5. 函数签名修复
- `get_messages_from_deepagents(session_id)` - 只需要 1 个参数
- `_build_messages_from_map(request_map)` - 返回 `(messages, progress)` tuple
- `_extract_preview_from_checkpoint(ckpt_dict)` - 期望 dict 输入，messages 是 dict 列表

#### 6. 返回值修复
- `get_session_metadata` 可能返回 `None` 或 `{}`
- `_extract_preview_from_checkpoint` 可能返回 `None`
