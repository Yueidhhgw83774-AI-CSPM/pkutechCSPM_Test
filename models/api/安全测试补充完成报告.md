def test_session_id_jwt_signature_tampering(self):
    """JWT署名改ざん検出 (OWASP A07)"""
    # 测试被篡改的 JWT 签名
    tampered_jwt = "eyJ...TAMPERED_SIGNATURE"
    request = Base64TextRequest(
        filename="test.txt",
        file_content_base64=base64_content,
        session_id=tampered_jwt

### 5. API-SEC-13: Unicode 规范化攻击

```python
def test_unicode_normalization_attack(self):
    """Unicode正規化攻撃 (OWASP A03)"""
    # 外观相似的字符（Homograph attack）
    confusable_filenames = [
        "аdmin.txt",  # 西里尔字母 'a'

        "gооgle.txt",  # 西里尔字母 'o'
        "аррӏе.txt"  # 多个西里尔字母
    ]
```

**验证内容**:
- ✅ 测试 5 种 homograph 攻击
- ✅ 警告：需要 Unicode 规范化
