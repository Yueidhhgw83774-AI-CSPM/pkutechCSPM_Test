## 🎨 设计模式识别

### 1. 单例模式 (Singleton)
```python
_checkpointer = None  # 全局缓存

async def get_async_checkpointer():
    global _checkpointer
    if _checkpointer is not None:
        return _checkpointer  # 返回缓存实例
    # ... 创建新实例
    _checkpointer = checkpointer
    return _checkpointer
```

**优点**:
- 避免重复创建昂贵的对象
- 确保全局一致性

### 2. 优雅降级 (Graceful Degradation)
```python
try:
    # 尝试使用 PostgreSQL
    return PostgresSaver()
except Exception as e:
    logger.error(f"初始化失败: {e}")
    logger.warning("回退到 MemorySaver")
    return MemorySaver()  # 降级方案
```

**优点**:
- 服务不会因为外部依赖失败而崩溃
- 提供基本功能保障

### 3. 策略模式 (Strategy)
```python
storage_type = os.getenv("LANGGRAPH_STORAGE_TYPE", "").lower()

if storage_type == "memory":
    return MemorySaver()
elif storage_type == "postgres":
    return PostgresSaver()
elif storage_type == "opensearch":
    return OpenSearchSaver()
```

**优点**:
- 易于扩展新的存储类型
- 运行时动态选择策略
